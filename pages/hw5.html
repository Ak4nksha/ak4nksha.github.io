Bigfoot Sightings by State

This visualization displays the top 20 U.S. states with the highest frequency of Bigfoot sightings. I chose a bar chart because it effectively compares a quantitative value (count of reports) across nominal categories (states). To ensure the chart was readable, I performed data transformations in the specification: I standardized the state names to uppercase to merge inconsistencies, filtered out missing values, and used a window transform to rank the states by count, keeping only the top 20.

Yearly Sighting Trends

This second chart illustrates the change in Bigfoot report frequency over time. I used a line mark to show the temporal trend, with the year on the x-axis and the number of reports on the y-axis. The interactivity I added is a dropdown menu that allows the user to filter the entire dataset by a specific state. This interactive element makes the visualization more useful by letting users isolate local trends (like a spike in Ohio) that might be hidden in the aggregate national data.
